A datagrid is simply a table. You define your columns and cells and you can automatically filter (by data type) and paginate over each column for larger numbers of records.

The data grid in Contember allows you to display data from different entities, link to them and freely modify the data display in the cell.

## Cells

Data grid in Contember supports these cells:

<ul>
    <li>
        <a style={{ fontSize: '120%', fontWeight: 'bold' }} href="#boolean-cell">Boolean cell</a><br />
        The <code>BooleanCell</code> renders yes/no for boolean values. And filters by them + N/A.
    </li>
    <li>
        <a style={{ fontSize: '120%', fontWeight: 'bold' }} href="#date-cell">Date cell</a><br />
        The <code>DateCell</code> renders date values in a human readable format and filters by choosing from - to.
    </li>
    <li>
        <a style={{ fontSize: '120%', fontWeight: 'bold' }} href="#enum-cell">Enum cell</a><br />
        The <code>EnumCell</code> renders enum values in a human readable format and filters by the enum values.
    </li>
    <li>
        <a style={{ fontSize: '120%', fontWeight: 'bold' }} href="#generic-cell">Generic cell</a><br />
        The <code>GenericCell</code> is useful to render any data you want in this cell.
    </li>
    <li>
        <a style={{ fontSize: '120%', fontWeight: 'bold' }} href="#has-many-select-cell">Has many select cell</a><br />
        The <code>HasManySelectCell</code> renders a all the values of the related entity and filters by them.
    </li>
    <li>
        <a style={{ fontSize: '120%', fontWeight: 'bold' }} href="#has-one-select-cell">Has one select cell</a><br />
        The <code>HasOneSelectCell</code> renders a single value of the related entity and filters by them.
    </li>
    <li>
        <a style={{ fontSize: '120%', fontWeight: 'bold' }} href="#number-cell">Number cell</a><br />
        The <code>NumberCell</code> renders numbers and add filters for numbers (equals, greater than, smaller than)
    </li>
    <li>
        <a style={{ fontSize: '120%', fontWeight: 'bold' }} href="#text-cell">Text cell</a><br />
        The <code>TextCell</code> renders text values and filters by them (contains, doesn't contain, matches exactly, starts with, ends with).
    </li>
</ul>
