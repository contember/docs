# <span className="version">Interface 1.2+</span> Layout overview

:::note
`LayoutKit`, `Slots` and `Directives` are available in Contember Interface version 1.2 and are meant to be used together with [`Scopes`](/reference/admin/pages/scopes).
:::

Contember comes with built-in components to help you build layouts of your projects easily:

- [Layout Kit](/reference/admin/layouts/layout-kit) — components to build your application layout:
	- Frame — your application frame
	- Bars — application bars e.g. headers, footers for tabs, actions of title
	- Content panels — panels to render main content of the page
	- Sidebar panels and toggle buttons — collapsible panels with, e.g. navigation
- [Slots](/reference/admin/layouts/slots) — components to render content in predefined places of the layout
- [Directives](/reference/admin/layouts/directives) — components to control your app from within any rendered page
